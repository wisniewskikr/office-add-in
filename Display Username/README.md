# Display Username

## Description
An Excel Office Add-in that displays the name of the currently logged-in Microsoft account user.
Built with React, TypeScript, Fluent UI v9, and Office.js.

## Installation
1. Clone the repository and navigate to the `Display Username` folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the dev server and sideload the add-in into Excel.
   - This requires Office Add-in dev certificates (`office-addin-dev-certs`).

## Usage
1. In Excel, click the **Demo** tab in the ribbon.
2. Click **Display Username**.
3. In the task pane that opens, click the **Display Username** button.
4. Your Microsoft account display name will appear below the button.
